Get alerted when your website go down with Monitoscope.

== Description ==
Monitoscope is a webservice that monitors websites and alerts in case of access failure, it allows up to monitor 1 free website.

== Installation ==

1. Download monitoscope.zip from the "download" link on the web page where you're viewing this.
2. Decompress the file contents.
3. Upload the monitoscope folder to your WordPress plugins directory (/wp-content/plugins/).
4. Activate the Monitoscope plugin from the WordPress back-office.
5. Sign-up for free on www.monitoscope.com
6. Get your API key and set it up on your Wordpress website
